# Logo 图片放置说明

## 放置位置

请将您的公司 Logo 图片放在此目录下：

### 导航栏 Logo
- **文件名**: `logo.png`
- **路径**: `static/images/logo.png`
- **建议尺寸**: 高度 40-50px，宽度自适应（保持原始比例）
- **格式**: PNG（支持透明背景）或 SVG

### Favicon（浏览器标签页图标）
- **文件名**: `favicon.png`
- **路径**: `static/images/favicon.png`
- **建议尺寸**: 32x32px 或 16x16px
- **格式**: PNG 或 ICO

## 示例文件结构

```
static/
└── images/
    ├── logo.png          # 导航栏Logo
    └── favicon.png       # 浏览器图标
```

## 注意事项

1. 如果 Logo 文件不存在，网站会显示文本 "OmniPok" 作为备用
2. Logo 图片建议使用透明背景的 PNG 格式，效果更好
3. 如果使用 SVG 格式，可以保持清晰度且文件更小
